# GitPusher
A simple bash script to make it easier to work with the Github and Gitlab

# Install
ArchLinux:</br>
`yaourt -S gitpusher`

# First
run this commands:</br>
`git config --global user.email 'YourEmail@domain.com'`</br>
`git config --global user.email 'YourEmail@domain.com'`</br>


# Run
Run in directory:</br>
`gitpusher`
